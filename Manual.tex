$$ f^{(1)}(x) = ( 2 \cdot ( ( x - n )^{ ( 2 - 1 ) } ) ) \cdot ( 1 - 0 ) $$

$$ f^{(1)}(x) = 2 \cdot ( x - n ) $$

Вот только скажите, что не очевидно следующее выражение:


$$ f^{(2)}(x) = 2 $$
